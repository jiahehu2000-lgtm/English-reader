# IELTS Reader Vocab

A single-page IELTS reading tool for importing text/PDF articles, clicking words for Chinese translations, saving vocabulary, tracking word frequency, exporting vocabulary, and running browser-side OCR for scanned PDFs.

## Use Locally

Open `index.html` in a browser.

## Publish With GitHub Pages

Use GitHub Pages with the repository root as the Pages source. The site entry file is `index.html`.

## Notes

- Vocabulary data is stored in each user's browser local storage.
- PDF OCR runs in the user's browser through Tesseract.js.
- The page is a tool only. Do not publish copyrighted IELTS test PDFs or extracted full test text unless you have the rights to do so.
